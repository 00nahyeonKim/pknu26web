# CS 기초 이론

## CDN(Content Delivery Network, 콘텐츠 전송 네트워크)

```
 지리적으로 분산된 여러 서버에 웹 콘텐츠(이미지, 영상, HTML 등)를 복사하여 저장한 뒤, 사용자에게 가장 가까운 서버에서 빠르게 데이터를 전송하는 시스템입니다. 웹 페이지 로딩 속도 향상, 원본 서버 부하 감소, 높은 가용성 및 DDoS 방어 등의 보안 강화 효과를 제공
```

## SVB(Scalable Vector Graphics)란?

```
GIF, JPEG, WBMP 및 PNG와 같은 웹용 비트맵 이미지 포맷은 픽셀 격자를 사용하여 이미지를 나타냅니다. 결과로 생성된 파일은 크기가 커져서 단일 해상도(흔히 저 해상도)로 제한되기 쉽고 웹에서 많은 대역폭을 소비합니다.

반면에 SVG는 이미지를 모양, 패스, 텍스트, 필터 효과 등으로 나타내는 벡터 포맷입니다. 결과로 생성되는 파일은 크기가 작고 웹 상에서, 인쇄할 때 그리고 리소스가 제한되어 있는
휴대용 장치에서도 고품질의 그래픽을 제공합니다.

사용자는 선명도, 상세도 또는 명확성을 잃지 않고도 화면에서 SVG 이미지를 확대하여 볼 수 있습니다.

또한 SVG는 텍스트와 색상 지원이 뛰어나므로 lllustrator 대지에 나타나는 이미지 그대로 사용자에게 보여줄 수 있습니다.
```
